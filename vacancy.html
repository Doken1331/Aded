  • Experience in commercial development (apart from training)

  • Knowledge of layout technologies - Mobile First and Desktop First
  • HTML5, CSS3, SСSS knowledge (being able to design a project architecture using this tool). Experience with Bootstrap
  • Ability to work with flexbox layout and grid layout, understanding how it works
  • Excellent skills with JavaScript and jQuery
  • Work experience with various jQuery plugins
  • Knowledge of Gulp, Webpack or any similar bundler
  • Work experience with git (having worked with git in the console, knowledge of basic commands, their use in commercial development, knowledge of GitFlow)
  • General knowledge of how the Backend works
  • Willingness to learn, take on new tasks, get to the bottom of things and dive into new challenges
  • It would be great if you could showcase your code, send a link to Github in your cover letter

It would be an advantage if you:

  • Have coded for CMS (Bitrix, Wordpress or something else)
  • Knowledge of JavaScript standards (ES6 and higher)
  • Knowledge of Zurb Foundation framework

On our part, we'll provide:

  • Worldwide Remote work
  • Full-time employment, with a convenient work schedule (negotiated individually with each candidate)
  • The opportunity to grow, learn by doing, explore new layout and programming horizons
  • Paid overtime.

Our hiring process

1

Apply for a job

And we will send you a response within three business days.

2

We will send you a test task

It will help us evaluate your information-searching skills and help you learn more about our activities.

3

Interview

If you complete the task successfully, then we will call you. You will get a response regardless of the results.

4

Internship

This is a two-week training course for you to learn the basics of the services that you will find useful in the future. During this time, you will not have any actual work tasks, only training.

5

Welcome aboard!

And the fun is just starting.
